### Key Features

1. Users can login using their Google account.
2. They can request registration as a borrower or manager.
3. After approval by the admin, the user gets access to their dashboard.
4. The admin can also suspend a user from the website as needed.
5. Only active borrowers will be able to apply for a loan.
6. After submitting a loan application, a fixed payment of $10 must be made as a loan processing fee.
7. Stripe must be used as the payment gateway for making payments.
8. After the user pays the loan processing fee, the loan will appear as a pending loan on the manager's dashboard.
9. The manager can verify the information and then approve or reject the loan.
10. After approval or regection, the updated status will be shown on the user's dashboard.
